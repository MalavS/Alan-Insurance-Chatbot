# 🤖 Alan Insurance AI Advisor

**Production-ready insurance recommendation app** built for Alan Insurance Canada internship application. Converts user clicks into personalized Bear/Marmot plan recommendations in seconds.


**One-click qualification:**
- 👤 Individual (Employed → Marmot | Self-Employed → Bear)
- 🏢 Business (1-49 → Bear Group | 50-500 → Marmot Group)
- ⚙️ Custom employee count input

## ✨ **Features**
- **Beautiful dark UI** - Professional Alan branding
- **Smart product logic** - Perfect Bear/Marmot matching
- **No external dependencies** - Pure Python + Tkinter
- **Production-ready** - Smooth animations, error handling
- **Sales-focused CTAs** - "Book demo" integration ready

# No installs needed (uses built-in Tkinter)
